# Calculator v3 - Modern Web Calculator

A beautiful, modern calculator web application with gradient UI and full keyboard support.

## 🎨 Features

- **Modern Gradient UI**: Beautiful purple/pink gradient design
- **Full Calculator Functions**: Addition, subtraction, multiplication, division
- **Keyboard Support**: Type directly or click buttons
- **Responsive Design**: Works perfectly on desktop and mobile
- **Error Handling**: Graceful handling of division by zero
- **Smooth Animations**: Hover effects and button feedback

## 📁 Files

- `index.html` - Calculator structure and layout
- `style.css` - Gradient styling and responsive design
- `script.js` - Calculator logic and keyboard controls

## 🚀 Quick Deploy to GitHub Pages

Run this command from the project directory:

```bash
bash deploy_calculator_v3.sh
```

Or use Python:

```bash
python3 ../deploy_calc_v3_final.py
```

## 🌐 Live URL

After deployment: **https://natovichat.github.io/calculator-v3/**
