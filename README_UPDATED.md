# Calculator v3 - Modern Web Calculator

A beautiful, modern calculator web application with a gradient purple theme.

## 🌐 Live Demo

**URL:** https://natovichat.github.io/calculator-v3/

## ✨ Features

- **Basic Operations**: Addition, Subtraction, Multiplication, Division
- **Decimal Support**: Handle decimal numbers
- **Clear & Delete**: C button clears all, DEL removes last digit
- **Previous Operand Display**: See your calculation history
- **Keyboard Support**: Use your keyboard for faster input
- **Division by Zero Protection**: Alerts when attempting to divide by zero
- **Number Formatting**: Displays numbers with locale-aware commas (e.g., 1,000)
- **Modern UI**: Beautiful gradient background (purple → violet)
- **Responsive Design**: Works perfectly on mobile and desktop
- **Smooth Animations**: Professional hover effects and transitions

## 🎨 Design

- Purple/pink gradient background
- Glass-morphism calculator card
- Color-coded buttons:
  - Gray gradient for numbers
  - Blue gradient for operators  
  - Pink gradient for equals
  - Orange gradient for clear
- Smooth hover and click animations
- Mobile-responsive layout

## 🎹 Keyboard Shortcuts

- **0-9**: Number input
- **. (period)**: Decimal point
- **+, -, *, /**: Operations
- **Enter or =**: Calculate result
- **Escape**: Clear all
- **Backspace**: Delete last digit

## 📱 Responsive

The calculator automatically adapts to different screen sizes:
- Desktop: Large, comfortable buttons
- Mobile: Touch-friendly interface with adjusted sizing

## 🛠️ Technology Stack

- Pure HTML5
- CSS3 (Gradients, Flexbox, Grid)
- Vanilla JavaScript (ES6+ Class syntax)

## 📂 Files

- `index.html` - Calculator structure and layout
- `style.css` - Modern styling with gradients and animations
- `script.js` - Calculator logic and keyboard support
